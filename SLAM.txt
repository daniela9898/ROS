For SLAM, first install:
    sudo apt-get install ros-noetic-gmapping
Then execute the following command:
    roslaunch turtlebot3_slam turtlebot3_slam.launch
This will start the SLAM node and the RViz window. You can now drive the robot around to create a map of the environment.

It creates the following topics:
    /initialpose
    /joint_states
    /map
    /map_metadata
    /map_updates
    /move_base_simple/goal
And prints the following information:
    Registering Scans:Done
    update frame 342
    update ld=6.80818e-07 ad=2.02541e-05
    Laser Pose= 0.333968 0.289335 2.81937
    m_count 114
    Average Scan Matching Score=358.522
    neff= 100
